# Skill File Reference

The skill file is generated at `.claude/skills/rcli/SKILL.md` by `rcli init`.

## Purpose

Instructs Claude Code how to interact with the rcli database:
- Always use `--format json` for programmatic access
- Run `rcli init` if `.rcli/` doesn't exist
- Documents all commands with examples
- Provides workflow guidelines

## Format

The skill file uses the new Claude Code skill format with YAML frontmatter containing `name`, `description`, and `user-invocable` fields. This allows Claude to auto-discover the skill when users discuss requirements or decisions.

## Regeneration

The skill file is regenerated on `rcli init`. It can also be manually regenerated by running init again.
